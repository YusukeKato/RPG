## Windows以外で遊びたい！

windowsでしか動作しない理由は、プログラム中で使っているwindows依存コードが原因なので

以下の対策をすることで動作すると思われる

`system ("cls");`

を削除する（コメントアウトする）

## 文字化け

文字化けが起こる場合は文字コードが原因なので

UTF-8に変換することでコンパイル＆実行できるようになるはず
